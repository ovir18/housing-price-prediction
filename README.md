# 🏠 Housing Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts house prices based on various property features using Machine Learning.  
It demonstrates a complete end-to-end data science workflow including data preprocessing, feature encoding, model training, and evaluation.

---

## 🎯 Objective
To build a regression model that can accurately predict house prices based on features like area, bedrooms, bathrooms, and other property attributes.

---

## 📊 Dataset Information
The dataset includes the following features:

- Area
- Bedrooms
- Bathrooms
- Stories
- Mainroad access
- Guestroom availability
- Basement
- Hot water heating
- Air conditioning
- Parking spaces
- Preferred area
- Furnishing status

Target:
- Price of the house

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

---

## 🤖 Machine Learning Model
- Extra Trees Regressor (Ensemble Learning)
- Train-Test Split (80/20)
- Feature Encoding for categorical variables

---

## 📈 Model Performance

- Mean Absolute Error (MAE): ~1,000,000+
- R² Score: ~0.62

---

## 📊 Visualizations
- Feature Importance Chart
- Actual vs Predicted Price Scatter Plot

---

## 🚀 Key Steps
1. Data Loading
2. Data Cleaning & Encoding
3. Feature Selection
4. Model Training
5. Evaluation
6. Visualization

---

## 📌 Project Insights
- Area and location-related features strongly influence house prices
- Ensemble models perform better than simple linear regression
- Data quality limits maximum achievable accuracy
